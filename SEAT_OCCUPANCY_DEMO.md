# 智能座位占用系統演示

## 📊 系統特點

### 1. 確定性隨機算法
- 相同航班 + 相同日期 + 相同艙位 = 相同的座位占用
- 每天凌晨0點自動更新
- 刷新頁面不會改變當天的座位分配

### 2. 動態占用率計算

#### 基礎占用率：50-70%
```
平日工作日：約 55% 占用
```

#### 周末加成：+10-20%
```
周六、周日：約 65-75% 占用
```

#### 節假日加成：+15-25%
```
元旦、春節、勞動節、國慶、聖誕：約 75-85% 占用
```

#### 旺季加成：+10-15%
```
暑假（7-8月）：約 70-80% 占用
寒假（12-2月）：約 75-85% 占用
國慶黃金周（10月）：約 80-90% 占用
```

## 💡 特殊規則

1. **疊加效應**
   - 周末 + 節假日 = 更高占用率
   - 節假日 + 旺季 = 接近滿座

2. **熱門航線**
   - HKG → SIN（香港-新加坡）
   - HKG → NRT（香港-東京）
   - HKG → FRA（香港-法蘭克福）

3. **時段因素**
   - 晚上航班通常更滿（模擬商務客）
   - 早晨航班相對較空

## 🔬 技術實現

### 哈希算法
```typescript
種子 = hash(航班ID + 日期 + 艙位)
隨機數生成器 = SeededRandom(種子)
```

### 座位選擇
```typescript
占用數量 = 總座位 × 占用率
for (i = 0; i < 占用數量; i++) {
  隨機選擇一個可用座位
  標記為已占用
}
```

## 📈 真實性模擬

### 符合航空業規律
- ✅ 周末和節假日客流量增加

### 一致性保證
- ✅ 同一航班同一天的座位固定
- ✅ 不會因為刷新頁面改變
- ✅ 多個用戶看到相同的座位狀態
