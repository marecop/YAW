## 🔧 技术实现

### 使用方法

所有页面都使用 `useLanguage` Hook 来访问翻译：

```tsx
'use client'

import { useLanguage } from '@/contexts/LanguageContext'

export default function MyPage() {
  const { t } = useLanguage()
  
  return (
    <div>
      <h1>{t('page.title')}</h1>
      <p>{t('page.description')}</p>
      <button>{t('common.save')}</button>
    </div>
  )
}
```

### 语言切换

用户可以通过页脚的语言选择器切换语言：
- 🇭🇰 粤语繁体 (zh-hk)
- 🇺🇸 English (en)
- 🇨🇳 简体中文 (zh-cn)
- 🇩🇪 Deutsch (de)

语言偏好会保存在浏览器的 `localStorage` 中。

## 📝 翻译键命名规范

### 格式
```
<类别>.<功能>
```

### 示例
```json
{
  "auth.login": "登入",
  "auth.loginTitle": "登入您的帳戶",
  "flights.searchResults": "航班搜索結果",
  "home.yellowExperience": "黃色航空體驗",
  "common.loading": "載入中..."
}
```

### 类别列表
- `site.*` - 网站基本信息
- `nav.*` - 导航栏
- `hero.*` - 英雄区域
- `search.*` - 搜索功能
- `flight.*` - 航班信息
