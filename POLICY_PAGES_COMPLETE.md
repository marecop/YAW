## 🎨 设计特色

### 1. 统一的视觉风格
- ✅ **黄色警告框**: 所有页面顶部都有黄色边框的重要提示框
- ✅ **面包屑导航**: 首页 → 当前页面
- ✅ **标题层级**: h1 (页面标题) → h2 (章节标题) → h3 (子标题)
- ✅ **更新日期**: 显示最后更新日期

### 2. 品牌颜色应用
- **黄色** (`ya-yellow`): 警告框、链接、按钮
- **灰色** (`gray`): 正文、背景
- **白色**: 卡片、内容区域

### 3. 图标使用
- ✅ 警告图标（黄色感叹号）
- ✅ 对号图标（行李页面）
- ✅ 文件图标（行李页面）
- ✅ 邮件和电话图标（联系方式）

### 4. 交互元素
- **按钮**: 圆角、hover效果、黄色主题
- **链接**: 黄色文字、hover变深
- **标签页**: 底部边框指示器（行李页面）

## 🌍 多语言支持

### 集成的翻译功能
所有页面都使用 `useLanguage` hook：

```tsx
import { useLanguage } from '@/contexts/LanguageContext'

const { t } = useLanguage()

// 使用翻译
<h1>{t('baggage.title')}</h1>
<p>{t('baggage.subtitle')}</p>
```
